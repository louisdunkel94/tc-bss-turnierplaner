In 3 Schritten zum Turnier

So läuft ein typischer Turniertag ab.

1
Turnier erstellen
Im Dashboard auf „Neues Turnier" klicken. Name, Datum, Spielmodus und Anzahl der Plätze festlegen. Optional eine Vorlage aus einem früheren Turnier laden.
2
Spieler eintragen & losen
Spielernamen eingeben oder aus der Vereins-Mitgliederliste per Chip hinzufügen. Dann „Turnier starten" — die erste Runde wird automatisch ausgelost.
3
Ergebnisse eintragen
Nach jeder Runde Ergebnis antippen — Tabelle und Statistiken aktualisieren sich sofort. Runde für Runde bis zum Finale, dann die animierte Siegerehrung starten.
